# Poppins Font Files

Download the Poppins font files from Google Fonts and place them in this directory.

## Required Font Files:
1. `Poppins-Regular.ttf`
2. `Poppins-Medium.ttf`
3. `Poppins-SemiBold.ttf`
4. `Poppins-Bold.ttf`
5. `Poppins-Light.ttf`

## Download Instructions:

### Option 1: Google Fonts (Recommended)
1. Go to: https://fonts.google.com/specimen/Poppins
2. Click "Download family"
3. Extract the ZIP file
4. Copy the required .ttf files to this `fonts` folder

### Option 2: Direct Download Links
You can also download individual font weights from:
https://github.com/itfoundry/Poppins/tree/master/products

## After Adding Fonts:
Run `npm install` in the Mobile folder to ensure expo-font is installed, then restart the app.
